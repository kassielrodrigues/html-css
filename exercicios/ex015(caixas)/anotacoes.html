Modelos de Caixa HTML-CSS

Box Model HTML5-CSS3 pesquisar

Quando se tem uma pagina html tudo oq é mostrado é com caixas
Pode conter uma caixa dentro da outra, o termo para se referir é ANINHAMENTO
Uma caixa contém um tamanho, width = largura | height = altura
Uma caixa contém uma borda, em CSS chame de border
Para impedir que a borda fique "colada" no conteudo, use o paddingm, o padding pode ser usado em uma direção ou em todas
Pode usar o margin, para criar uma margem fora da caixa

RESUMO:
width = largura
height = altura
border - cria uma borda na box
padding - cria um espaçamento interno na box
margin - cria um espaçamento externo na box
outline - cria uma borda dentro do margin

Tipos de Caixa

Box Level & Inline level diferença basica

Exemplo: Supomos que exista um texto que acaba no meio da tela
Box-level será criado embaixo da texto e sempre ocupa totalmente a largura da tela (width)
A largura do box Level é sempre 100% do viewport

Exemplo: Supomos que exista um texto que acaba no meio da tela
Inline-level ao contrário do box level, a caixa será criada ao lado do texto
no tamanho necessario para conter a informação
