Principais formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a <b> (sem semantica)

Não ter semantica significa não ter forma é simplesmente negrito

Nesta frase temos um termo em destaque usando a tag <strong> (com semantica)

Itálico ou ênfase

Nesta frase temos um termo itálico usando a tag <i> (não semantico)

Nesta frase temos um termo em ênfase usando a tag <em> (semantico)

Forma e sentido

Forma: CSS
Sentido: Html

Html é apenas colocar os objetos na tela, quem posiciona e organiza são as CSS

A Html é fundamental
