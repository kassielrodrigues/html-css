Pagina 2 - Target

Por padrao sempre que se abre um HyperLink, o site de destino abre na mesma janela, esse comportamento é usado quando o visitante continua no nosso site

O ideal é sempre que tiver um link que redireciona a página a outra página, que não pertence ao seu voce é usar os paramentros target="_blank" rel="external" para evitar que o seu site seja "perdido"

Tipos de Target

_blank=""

_blank vai abrir o link em outra janela em branco
_self vai abrir o link na mesma janela (padrao)
_top vai desfazer todos os frames/janelas e abrir o destino no navegador completo
_parent similar ao uso _top em uma referencia a janela mãe

Testando a target: Nasa _self
Testando a target: Nasa _blank
Testando a target: Nasa _top
Testando a target: Nasa _parent

Link que levam para outros arquivos dentro do site são chamados de link interno ou link local

Link que levam para outros sites são chamados de link externo.
Nesse caso devemos escrever a URL completa inclusive no começo https:// +Link

Estudar rel=""nofollow e dofollow"
