Grouping tags (tags de agrupamento)

Duas tags de agrupamento são a DIV e o SPAN
div é uma caixa do tipo BoxLevel e span é do tipo InlineLevel

No html-4 se criava uma div para ser o cabeçalho, uma para ser o conteúdo e uma para o rodapé
No html-5 existe a tag header(cabeçalho), main(conteúdo) e footer(rodapé)

O header, main e footer não deixam de ser uma div, mas deixam a script mais organizada e são tags semanticas
